## 未リリース

- 2026-07-07 08:56 保存を明示方式に変更（保存ボタン / Ctrl+S）。保存のたびに履歴が残り、「履歴」から差分表示と過去版の読み込みができる。未保存の編集は自動でドラフト退避され、次回オープン時に復元できる。
- 2026-07-07 08:56 サイドバーの各行に「…」メニューを追加。ワークスペース / ドキュメントの名前変更と削除ができる。

- 2026-07-07 08:35 ライブラリ機能を追加。データの保存先フォルダ（DB・画像・RAG）を「ライブラリ」として新規作成・切り替えできる。上部バー左のライブラリメニューから操作。

- 2026-07-07 08:24 RAG を追加。資料の取り込み（API / 一括投入 CLI）と hybrid 検索（意味検索 + 全文検索）ができ、執筆支援の「RAG」チェックで検索結果を参照した生成が可能に。

- 2026-07-07 08:14 画面上部にモデルバーを追加。アプリ内から LLM モデルの選択・起動・切替・停止ができ、状態（未起動/起動中/稼働中）が常時表示される。start-llm.bat を手動実行しなくてもよくなった。

- 2026-07-07 08:05 分割ビュー校正を追加。2 段落以上の選択（または「文書全体を校正」）で左右比較ビューが開き、段落ごとに採用/スキップ、一括採用ができる。

- 2026-07-07 07:58 執筆支援を追加。「続きを生成」と「指示からセクション生成」ができ、結果をカーソル位置に挿入できる。LLM の思考モードを無効化し応答を高速化（校正 0.7 秒前後）。

- 2026-07-07 07:47 インライン校正を追加。選択範囲を Gemma 4 で校正し、diff 表示から採用/破棄できる。LLM サーバは `start-llm.bat` で起動。

- 2026-07-07 07:37 `start.bat` を追加。ダブルクリックで backend（別ウィンドウ）と Vite + Electron を一括起動。

- 2026-07-07 07:32 エディタ基盤を実装。ワークスペース / ドキュメント管理（サイドバー）、SQLite への自動保存、画像のペースト / ドロップ挿入、FastAPI backend（`npm run backend` で起動）。
- 2026-07-07 07:23 アプリ雛形を作成（Electron + Vite + React + TypeScript、TipTap v3 エディタの基本表示）。`npm run dev` / `npm run build` / `npm start` が使用可能に。
